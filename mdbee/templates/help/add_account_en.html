Accounts 



Dental Bee accounts have two main types:

  • Headquarters – an account that is the main “bill-to” account.
  • Subsidiaries – an account that has a headquarters account that it is associated to

The Accounts table is the main table for managing accounts, subscriptions, users and devices in Dental Bee.  



 The NetsuiteId field is populated with the NetSuiteId in the MDBee NetSuite application. The CorporateId is determined by the system based on the account record:

  • If the NetSuite account is a headquarter account, the CorporateId is set to the same value as the NetSuiteId

  • If the NetSuite account is a subsidiary account, The CorporateId is set to the headquarter record’s NetSuiteId.

  • If the Netsuite account is a subsidiary account that is also a headquarter account, the CorporateId is the Netsuite Id is the subsidiary that it is related to.
